# Fluxo de Conversão – Autoescola Jotta

## Estrutura do Funil

### PÁGINA 1: Landing Page de Captura
*   **Objetivo:** Capturar leads interessados na promoção.
*   **URL:** `autoescolajotta.vercel.app`
*   **Ação:** Usuário preenche o formulário.

### PÁGINA 2: Thank You Page (Redirecionamento pós-formulário)
*   **Objetivo:** Levar o lead para o Grupo VIP no WhatsApp.
*   **URL:** `autoescolajotta.vercel.app/grupo-vip`
*   **Ação:** Usuário clica no botão para entrar no grupo.
